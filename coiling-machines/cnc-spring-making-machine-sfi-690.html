SFI-690

Balanced 6-axis CNC spring production

  • 6-axis CNC system with 6 servo motors for accurate control.
  • CNC control of spring diameter removes the traditional 5-axis CAM reduction mechanism.

Machine Features

Details extracted from the provided SFI-690 image.

The 6-axis CNC spring machine uses 6 sets of servo motors and abandons the traditional 5-axis CAM reduction mechanism, making operation easier to learn and understand. The CNC system directly controls spring diameter and reducing diameter. Five wire feed rollers help ensure the wire precision is not crushed and provide enough traction for stable wire feeding. The assist mandrel mechanism supports manual or automatic locking and is fixed by hydraulic jacking after lifting, which is easy to operate with small vibration and high incision precision. Straight cut, subtend cut, swing cut, and twist cut functions are complete. The wire feeding mechanism has a two-speed transmission system for high and low speed. The control system adopts the advanced 5G-3D (AW5000) touch screen control software with a remote Internet interface and 3D automatic programming template. After entering the required values, the system automatically generates the spring program, and the required spring can be completed with slight modification. It can be equipped with an image detector.
