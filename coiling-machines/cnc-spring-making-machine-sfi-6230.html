Machine Features

Details extracted from the provided SFI-6230 image.

The 6-axis CNC spring machine uses 6 servo motors and abandons the traditional 5-axis cam reducing mechanism. The CNC system directly controls spring diameter and reducing diameter for easy operation. Six sets of wire-feeding shafts reduce force to keep the steel wire precision without crushing. The auxiliary core cutter adopts hydraulic locking after lifting for easy operation with low vibration and high cutting precision. Straight cut, subtend cut, swing cut, and twist cut functions are supported. The 5G-3D (AW5000) touch screen control system includes a remote Internet interface and a 3D auto-programming template that generates spring programs automatically; it can be equipped with an image detector.
