Machine Features

The 11-axis CNC spring machine uses 11 sets of servo motors and abandons the traditional cam reduction mechanism. The CNC system controls the spring outer diameter and variable diameter, and the feeding system has 6 sets of servo motors optimized for the 6-axis function with total wire-feeding power up to 90 kw. It supports a single motor for small-diameter springs or 6 motors for high-speed synchronized feeding. The hydraulic core knife and locking structure improve stability, reduce vibration, and ensure high cutting precision. It supports straight, slubed, swing, and twist cutting functions. The 5G-3D (AW5000) touch screen system includes a remote internet interface and 3D auto-programming templates that generate spring programs automatically; it can be equipped with an image detector.
