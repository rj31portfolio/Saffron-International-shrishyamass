Machine Features

Details extracted from the provided SFI-960 image.

The 9-axis CNC spring machine uses 9 sets of servo motors. The coiling axis uses a one-piece structure to achieve stable horizontal and vertical 2D coil spring movement. The CNC system directly controls the outside diameter and spring diameter so adjustments are handled in the control system without changing the mechanical coiling axis slider. The auxiliary core cutter is servo controlled for precision and convenience, and the cutter and auxiliary core cutter can be moved together to maintain the optimal position when changing the spring diameter. Straight cutting, subtend cutting, swing cutting, and twisting cutting functions are supported. The 5G-3D (AW5000) touch screen control system includes a remote Internet interface and a 3D auto-programming template that generates spring programs automatically; it can be equipped with an image detector.
